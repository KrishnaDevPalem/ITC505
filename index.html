Lights Out Game

Click the cells to turn them on or off. Try to turn off all the lights!

Project Addendum

This Lights Out Game was developed with careful consideration of user interaction and game dynamics. The following describes the thought process and methods used in the development:

Thought Process:

  • User Engagement: The main goal was to create an engaging user experience. Clicking on cells to toggle lights on or off provides a simple yet interactive gameplay.
  • Challenge Factor: The difficulty was incorporated by designing the game to require users to strategically turn off all the lights. This adds a level of challenge and encourages critical thinking.

Methods Used:

  • HTML Structure: The HTML structure is designed to provide a clear and organized layout for the game, including a header, game container, and controls.
  • CSS Styling: The styles.css file is utilized for styling, ensuring a visually appealing and responsive design across various devices.
  • JavaScript Functionality: The game logic is implemented in the script.js file. It includes functions for resetting the game and handling user interactions to manipulate the lights.

This project aims to showcase a combination of HTML, CSS, and JavaScript to create an enjoyable and challenging game. Feel free to explore and enjoy the Lights Out Game!
